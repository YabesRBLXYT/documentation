API Documentation

Translate Service by Yabes

Base URL: https://yabes-desu.serv00.net/translate

Introduction

Welcome to the Translate Service API. This service enables effortless text translation between languages using various endpoints and parameters.

Note: This API is free to use and may take some time to process. Please be patient!

Server 1 - Main API

Endpoints

Available API endpoints:

  • GET: 
    /api?text=Hello&fr=en&to=id
  • POST: Send 
    text, fr, and to
    parameters to 
    /api

Parameters

  • text
    : The text to be translated (String).
  • fr
    : Source language code (e.g., "en").
  • to
    : Target language code (e.g., "id").

Response

The API returns a JSON object in the following format:

{
    "success": true,
    "translated_text": "Translated text here",
    "credit": "t.me/Yabes_Desu"
}

Server 2 - Alternative API

Faster, but the meaning is often wrong

  • GET: 
    /api2?text=Example&fr=en&to=id
  • POST: Send 
    text, fr, and to
    parameters to 
    /api2

Parameters Explained

  • text
    : The text to be translated. This is required and must be a string (e.g., "Hello").
  • fr
    : The source language code (e.g., "en" for English). This is required.
  • to
    : The target language code (e.g., "id" for Indonesian). This is required.
  • tr
    : (Optional) Specifies which translation result to display.
    • If 
      tr=1
      , the response will return the primary translation (usually the most accurate).
    • If 
      tr=2
      , the response will return the second translation (if available).
    • If 
      tr
      is not provided or the requested translation index is unavailable, it defaults to 
      tr=1
      .

Response Explained

The API response from Server 2 returns a JSON object structured as follows:

{
    "success": true,
    "translated_text": "Translated text from Server 2",
    "amount_translate": 2,
    "credit": "t.me/Yabes_Desu"
}
        
  • success
    : A boolean value indicating whether the translation was successful (true or false).
  • translated_text
    : The final translation result based on the requested `tr` parameter.
  • amount_translate
    : The total number of available translations for the input text. This value indicates how many unique translations are returned by the API.
  • credit
    : A string to credit the developer (in this case, "t.me/Yabes_Desu").

Notes

Important: If the 

tr
parameter is missing or requests a translation index that doesn't exist, the API defaults to returning 
tr=1
.

If 

amount_translate
is 1, it means only one translation is available for the given input. If it's greater than 1, you can use the 
tr
parameter to access additional translations.
